网络环境越来越差了，之前使用picgo+github作为图床的方案已经几乎不可用了。经过一番折腾，发现了使用PoweredBy.Cloud + OneDrive的方案。

PoweredBy.Cloud

PoweredBy.Cloud能将OneDrive或GoogleDrive转变为一个兼容S3 API的CDN。简单说就是可以使用OneDrive或GoogleDrive中的一个文件夹来作为存储空间，并生成这个空间内的文件的外链。这样就可以实现将OneDrive作为图床使用的目的了。

使用PoweredBy.Cloud很简单，只要先进入官网：https://poweredby.cloud/，并使用邮箱注册一个账号。再通过Dashboard -> ADD SITE来添加一个域名即可：

添加好域名后，还需要通过Dashboard -> Developer -> CREATE ACCESS KEY来创建一个资源访问Token。之后就可以使用这个Token来进行资源管理了。

AWS CLI

可以通过S3 API对PoweredBy.Cloud中的资源进行管理，详见官方文档。这里我们选用AWS CLI。
